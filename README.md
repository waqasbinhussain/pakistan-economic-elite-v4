# 🇵🇰 Pakistan Economic Elite Model (2013–2025)

A comprehensive economic analysis and modeling toolkit covering Pakistan’s macroeconomics, debt dynamics, and political tenure comparison.

---

## 📊 What’s Included

* 📈 Year-by-year macroeconomic data (GDP, inflation, debt)
* 🏛 Political tenure comparison (Nawaz, Imran, Shehbaz)
* 💰 Loan breakdown (IMF, World Bank, China, Gulf)
* 🔥 Lender heatmap (multi-year exposure)
* 🧠 Debt sustainability model
* ⚠️ Scenario simulator (FX shock, oil shock)
* ⚡ Circular debt tracking (power sector)
* 🗳 Election-year analysis
* 🏢 Provincial debt insights

---
